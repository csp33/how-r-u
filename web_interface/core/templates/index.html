What is HOW-R-U?

HOW-R-U is a system designed to help doctors track their patients' status between therapies. It consists on two parts: an interactive web interface where doctors can assign questions to the patients, view the retreived data, extract spreadsheets, etc. and a Telegram conversational agent which will ask the corresponding questions and register the responses.

Create your own questions or use other doctor's ones

Set a custom question's frequency

Generate custom spreadsheets with the retrieved data

View the answer's tendency with interactive graphics
